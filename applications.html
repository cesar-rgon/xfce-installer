Lista de aplicaciones del script

Listado de aplicaciones que puede seleccionar el usuario durante el proceso de instalación del script.



Categoría: Oficina
Página Web: http://www.abisource.com/

Procesador de texto ligero y gratuito similar a Microsoft® Word. Es adecuado para una gran variedad de tareas. Permite trabajar con más personas en un mismo documento a la vez. Está estrechamente integrado con el servicio web AbiCollab.net, el cual te permite almacenar documentos online, compartir documentos con tus amigos fácilmente y realizar conversiones de formato sobre la marcha.


Categoría: Accesorios
Página Web: https://launchpad.net/alacarte

Editor de menú simple para Gnome que usa la librería pyxdg. Permite crear y editar menús y entradas. Sigue las especificaciones de menú freedesktop.org por lo que debería funcionar con cualquier WM/DE que soporte la especificación.


Categoría: Internet
Página Web: http://amule.forumer.com/index.php

Programa de descargas de enlaces edonkey mediante el protocolo de comunicación P2P. Es el equivalente a Emule de Windows, siendo aMule multiplataforma. Permite los modos clásico y demonio. Clásico: Cliente gráfico que debe estar ejecutado para realizar las descargas. Demonio: Servidor de descargas por consola, además incorpora cliente gráfico que se conecta al demonio local o remotamente.


Categoría: Accesorios
Página Web: http://docs.xfce.org/xfce/xfce4-appfinder/start

Herramienta para encontrar y lanzar aplicaciones instaladas en el sistema y ejecutar comandos rápidamente. Puede ser usado para añadir rápidamente lanzadores al panel de Xfce al arrastrar un elemento desde la vista y soltarlo sobre el panel.


Categoría: Multimedia
Página Web: http://audacious-media-player.org/

Reproductor de audio open source descendiente de XMMS. Reproduce la música consumiendo pocos recursos de tu ordenador. Permite arrastrar y soltar carpetas de música y canciones individuales, buscar por artistas y albums en tu librería musical, crear y editar tu propia lista de canciones personalizada, escuchar CDs o música streaming de internet, tunear la música con el ecualizador gráfico o experimentar con LADSPAeffects. Presenta un interfaz GTK moderna y skins similares a el clásico Winamp. Se puede usar los plugins incluídos para buscar liricas a las canciones, para establecer alarma y más.


Categoría: Sistema
Página Web: http://bleachbit.sourceforge.net/

Herramienta open source que libera rápidamente espacio de disco y mantiene tu privacidad. Libera caché, borra cookies, limpia el historial de Internet, ficheros temporales, borra logs y otros residuos. Diseñado para sistemas Linux y Windows, limpia miles de aplicaciones tales como Firefox, Internet Explorer, Adobe Flash, Google Chrome, Opera, Safari y más. Además de borrar ficheros, incluye características avanzadas como desechar ficheros para prevenir ser recuperados y liberar espacio de disco para ocultar trazas de ficheros borrados por otras aplicaciones.


Categoría: Desarrollo
Página Web: http://bluefish.openoffice.nl/index.html

Editor potente destinado a programadores y desarrolladores web, con muchas opciones para elaborar sitios web, scripts y código de programación. Soporta muchos lenguajes de etiquetas y programación.


Categoría: Oficina
Página Web: http://calibre-ebook.com/

Aplicación gratuita y open source de manejo de biblioteca de e-book desarrollado por y para usuarios de e-books. Tiene una gran cantidad de características que se agrupan en las siguientes categorías: Gestión de biblioteca, conversión de e-book, sincronización con dispositivos e-book, descargar noticias de la web y convertir a formato e-book, visor de ebook, servidor de contenido para acceso online a tu colección de libros.


Categoría: Accesorios
Página Web: http://twotoasts.de/index.php/catfish/

Interfaz de búsqueda destinado a localizar y encontrar ficheros, con autocompletado de Zeitgeist. Las opciones avanzadas permite filtrado por fecha y tipo de fichero. La interfaz es ligera y simple, usando sólo GTK+.


Categoría: Sistema
Página Web: https://launchpad.net/lubuntu-software-center

Utilidad para explorar, instalar y borrar aplicaciones de Lubuntu. Te permite explorar e instalar miles de aplicaciones disponibles para Lubuntu. Puedes ver las aplicaciones disponibles por categorías o buscar rápidamente por nombre o descripción. Puedes además examinar las aplicaciones ya instaladas y eliminar aquellas que ya no necesitas más.


Categoría: Sistema
Página Web: https://launchpad.net/software-center

Utilidad para instalar y manejar software en Ubuntu. Puedes localizar e instalar fácilmente software y comprar aplicaciones comerciales. Puedes puntuar y valorar software, estas estadísticas son usadas para clasificar el software por popularidad. En la versión 5.2, puedes obtener recomendaciones basadas en el software que ya tienes instalado.


Categoría: Internet
Página Web: https://www.google.com/chrome

Navegador web rápido y gratuito. Desarrollado por Google, es un navegador muy popular que puede ser usado en tu ordenador, teléfono movil y tablet. Dispone de gran número de plugins y complementos que extienden su funcionalidad y seguridad.


Categoría: Internet
Página Web: http://www.chromium.org/Home

Navegador open source que proporciona una manera más segura, rápida y estable de experimentar la web a todos los usuarios de internet.


Categoría: Multimedia
Página Web: http://www.clementine-player.org/

Reproductor de música multiplataforma moderno que permite organizar una biblioteca de audio. Inspirado en Amarok 1.4, ofrece una interfaz rápida y fácil de usar para buscar y reproducir su música.


Categoría: Sistema
Página Web: http://www.compiz.org/

Es un gestor de composición OpenGL que usa GLX_EXT_texture_from_pixmap para decorar títulos de ventanas a objetos con textura. Tiene un sistema flexible de plugins y está diseñado para correr bien en muchas tarjetas gráficas. Mejora la experiencia de usuario al añadir efectos a tus ventanas, desde proyectar sombras a asombrosos efectos de escritorio como el Cubo o vista Expo. Puede actuar como gestor de ventanas, por lo que permite mover, redimensionar ventanas, cambiar de escritorio y ventanas, etc.


Categoría: Internet
Página Web: http://deluge-torrent.org/

Programa multiplataforma de descargas de enlaces torrent mediante el protocolo de comunicación P2P. Usa libtorrent en su frontal e incorpora multiples interfaces de usuario: GTK+, web y consola. Permite los modos clásico y demonio. Clásico: Cliente gráfico que debe estar ejecutado para realizar las descargas. Demonio: Servidor de descargas por consola, además incorpora cliente gráfico que se conecta al demonio local o remotamente. Tiene una gran colección de plugins que extiende funcionalidad al programa.


Categoría: Internet
Página Web: https://www.dropbox.com/

Servicio gratuito que te permite disponer de tus fotos, documentos y vídeos en cualquier parte, y compartirlos fácilmente. Dió sus primeros pasos en 2007, de manos de sus fundadores Drew Houston y Arash Ferdowsi, dos alumnos del MIT cansados de enviarse archivos por correo electrónico a sí mismos para trabajar desde diferentes ordenadores. A fecha de hoy, más de 100 millones de personas en todo el mundo utilizan Dropbox para tener sus cosas siempre a mano, compartirlas con su familia y amigos, y trabajar en proyectos en equipo.


Categoría: Oficina
Página Web: http://projects.gnome.org/evince/

Visor de documentos para múltiple formatos. El objetivo es integrar en una sola aplicación las funcionalidades de otros visores de documentos que existen en Gnome. Ha sido diseñado para soportar los siguientes formatos de fichero: PDF, Postscript, djvu, tiff, dvi, XPS, soporte SyncTex con gedit, libros de comics (cbr,cbz,cb7 and cbt).


Categoría: Internet
Página Web: http://projects.gnome.org/evolution/

Aplicación que gestiona las cuentas de correo, libreta de direcciones y calendario similar a Outlook de Windows para usuarios de escritorio Gnome.


Categoría: Accesorios
Página Web: http://fileroller.sourceforge.net/

Interfaz grágica para gestores de archivos como tar y zip. Con esta aplicación se puede: crear y modificar archivos, ver contenido de un archivo, ver y modificar un fichero contenido en el archivo, extraer ficheros de un archivo, guardar el archivo en diferente formato. Desde la v3.6.0 se puede usar libarchive para manejar algunos tipos de archivos. Los formatos soportados son: .7z, .ace, .alz, .ar, .arj, .cab, .cpio, .deb, .iso, .jar, .ear, .war, .lzh, .lha, .rar, .cbr, .rpm, .bin, .sit, .tar, .tar.gz , .tgz, .tar.bz , .tbz, .tar.bz2 , .tbz2, .tar.Z , .taz, .tar.lz , .tlz, .tar.lzo , .tzo, .tar.7z, .tar.xz, .zip, .cbz, .zoo, gzip, bzip, bzip2, lzip, lzop, rzip, xz.

Cliente gráfico open source para los protocolos FTP, FTPS y SFTP con muchas características. Soporta Windows, Linux, Mac OS X y más.

Hace uso del motor Gecko para renderizar las páginas webs que implementan los estándares actuales. Dispone de gran número de plugins y complementos que extienden su funcionalidad y seguridad.


Categoría: Accesorios

Calculadora basada en GTK 2 / GTK 3 con notación ordinaria / inversa, modo de fórmulas, diferentes bases numéricas (DEC, HEX, OCT, BIN) y diferentes medidas angulares (DEG, RAD, GRAD). Soporta punto flotante de cuádruple precisión y aritmética binaria de 112 bits.


Categoría: Multimedia
Página Web: http://www.gcstar.org/

Aplicación gratuita open source para gestionar colecciones de fichas. Se puede descargar automáticamente información detallada de cada elemento de internet y almacenar información adicional, tal como la localización o a quién se ha prestado. Se pueden realizar búsquedas filtradas de la colección por muchos criterios. Puede ser usado en los sistemas operativos donde esté disponible gtk2-perl (Linux y Windows).


Categoría: Sistema

Herramienta que realiza la instalación de ficheros deb. Permite instalar paquetes deb locales resolviendo e instalando sus dependencias.


Categoría: Desarrollo
Página Web: http://www.geany.org/

Entorno de desarrollo pequeño y ligero. Desarrollado para suministrar un IDE rápido que tiene pocas dependencias respecto a otros paquetes. Independiente del escritorio usado (KDE, Gnome, etc). Sólo requiere librerías GTK2. Algunas características son: resaltado de sintaxis, autocompletado de palabras claves, auto-finalización de etiquetas XML y HTML, consejos, soporte para muchos tipos de archivos: C, Java, PHP, HTML, Python, Perl, Pascal y otros, listas de símbolos, navegación del código, compilación y ejecución del código, gestión del proyecto, interfaz de plugins, etc.


Categoría: Oficina
Página Web: http://projects.gnome.org/gedit/

Editor de texto oficial de Gnome, simple, fácil y potente para propósito general. Características: Soporte para texto UTF-8, resaltado de sintaxis para varios lenguajes (C, C++, Java, HTML, XML, Python, Perl y otros), deshacer/rehacer, editar ficheros remotamente, revertir ficheros, soporte para preview e impresión, cortar/copiar/pegar, buscar y reemplazar, ir a línea específica, sangrado automático, justificación de texto, número de líneas, margen derecho, resaltado de línea actual, marcado de corchetes, ficheros de backup, fuentes configurables y colores, manual de usuario online, plugins.

Programa de manipulación de imágenes GNU. Tiene muchas funcionalidades. Puede ser usado como programa de dibujo, retocado de fotos, composición y creación de imágenes, sistema de procesamiento online, renderizado de imágenes, conversor de formatos de imágenes, etc. Dispone de plugins y extensiones para aumentar su funcionalidad. Su interfaz avanzada permite realizar las tareas de forma sencilla.


Categoría: Multimedia
Página Web: http://www.mplayerhq.hu

Interfaz gráfica de Gnome para el reproductor multimedia Mplayer. Reproduce muchos formatos de archivos de video: MPEG/VOB, AVI, Ogg/OGM, VIVO, ASF/WMA/WMV, QT/MOV/MP4, RealMedia, Matroska, NUT, NuppelVideo, FLI, YUV4MPEG, FILM, RoQ, PVA, soportado por codecs XAnim y librerías Win32. Se pueden reproducir películas VideoCD, SVCD, DVD, 3ivx, DivX 3/4/5, WMV y H.264. Soporte de muchos drivers gráficos, pantalla completa, visor de estado (OSD), subtítulos, controles por teclado, fuentes multi-idioma y formatos de subtítulos.


Categoría: Oficina

Aplicación open source gratuita escrita en Java, diseñado para simplificar la descarga de archivos de servidores como Rapidshare.com o Megaupload.com, no solo para usuarios con cuenta Premium sino también para los de cuenta gratuita. Ofrece descargas múltiples paralelas, reconocimiento de captcha, extracción automática de archivos, administración de contraseñas, etc. Soporta muchos sitios de encriptación de enlaces, de manera que el usuario solo tendrá que pegar los enlaces encriptados. Puede importar archivos DLC, CCF y RSDF.

Creación de formularios, creación e importación de tablas y consultas de multiples tipos de bases de datos, etc. Math: Editor de ecuaciones matemáticas, químicas, eléctricas o científicas con notación estándar.

Snapshot: Imágen de los datos de la fuente en un instante de tiempo específico. Revertir a estado anterior. Sincronización de directorios. Fuente y destino en conexiones remotas. Perfiles de configuración. Planificación de tareas. Log. Modo consola. Etcétera.
